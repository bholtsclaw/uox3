The UOX3 Command List
as of Tuesday, December 28, 2004

Key
  d: Decimal Number
  h: Hexadecimal Number
  s: String (Used for Sub-Commands)
  text: Text to send with the command
  on / off: Used to toggle settings
  nothing: Command that accepts params but can function without them

A
ACTION (Target Command)(h) Performs an animated action sequence
ADD (s s / s h h / h / nothing ) Adds a new item, npc, or opens the GM Menu if no parameters are provided.
    ITEM(s / h h): Adds item based on entry in items.dfn or specified hex code.
    SPAWNER(s / hh): Adds a spawn item based on entry in items.dfn or specified hex code.
    NPC(s / h h): Adds npc based on entry in npc.dfn or specified hex code.
    If a hex ID is provided it will add an object directly from the .mul files
    If nothing is specified, opens the Add Menu.
ADDACCOUNT (s s h) Adds an account first string is username, second is password, hex parameter is privs (if any)
ADDX (h / h h) Adds a new item to your current location
ADMINYELL (text) Admin Yell, Broadcasts message to all online with command level of 3 and above.
ALLMOVE (on / off) Enables or disables GM ability to pickup all objects
ANNOUNCE (on / off) Enables or disables announcement of worldsaves
AREACOMMAND Performs various functions on items in an area
    DYE (h): Sets all items in targeted area to specified colour.
    WIPE: Wipes all items in targeted area.
    INCX (d): Increments X value by specified amount.
    INCY (d): Increments Y value by specified amount.
    INCZ (d): Increments Z value by specified amount.
    SETX (d): Sets X value.
    SETY (d): Sets Y value.
    SETZ (d): Sets Z value.
    SETTYPE (d): Sets all items in targeted area to specified type.
    NEWBIE (d): Newbiefies all items in targeted area.
    SETSCPTRIG (h): Gives all items in targeted area specified trigger.
    MOVABLE (d): Sets all items in area to movable/unmovable.
 
B
BAN (Target Command) Bans the target from the server (not currently functional)
BOLT (Target Command) Plays the "bolt" effect on targeted character
BRIGHTLIGHT (h) Sets default daylight level
BUY (Target Command) Opens the "buyshop" gump for specified player
 
C
CNSYELL (text) Counsellor Yell - Announce a message to all counselors and above
CLEANUP Deletes all corpses and temporary moongates in the world.
CQ (s / nothing) Perform various tasks relating to the Counselor Queue
    NEXT: Goes to the next call in the queue.
    CLEAR: Clears the call you are currently on.
    CURR: Goes to the call you are currently on.
    TRANSFER: Transfers the current call to the GM queue.
    If nothing is specified, will display the Counselor Queue
COMMAND (s) Executes a trigger scripting command
CSTATS (Target Command) Opens a gump displaying the stats of the targeted character
 
D
DELETECHAR (Target Command) Deletes the targeted character
DARKLIGHT (d) Sets the default nighttime light level
DUNGEONLIGHT (d) Sets default dungeon light level
DUPE (d / nothing) Duplicates an item (parameter is how many copies to make)
DISCONNECT (d) Disconnects player connected to the specified slot
DYE (h / h h / nothing) Dyes an item the specified colour, or brings up the dye menu
DELID (h / h h) Deletes all items in the world with specified ID
 
F
FULLSTATS (Target Command) Sets all stats on targeted character to maximum
FORCEWHO Brings up an interactive listing of online users
FIX (d) Attempts to fix incorrect Z coordinates
 
G
GETLIGHT Displays the current light level
GMYELL GM Yell, Sends the message to all online GM's or above
GUARDS (on / off) Enables / Disables Town Guards globally
GMS Displays a list of all online GMs
GUMPMENU (d) Opens the specified Gump menu from the scripts
GMMENU (d) Opens the specified "Help" menu
GO (d d d / s h h h h / s d) Teleports you to a specified location
    CHAR (h h h h / d): Will teleport you to the character matching the provided serial or the connection slot.
    PLACE (d): Will take you to the specified place from locations.dfn
    If neither of the above are specified will take you to the x y z locations specified.
GCOLLECT Forces "Garbage Collection" routine to delete objects in the deletion queue
GLOW (Target Command) Gives glowing property to an object
GATE Creates a moongate using the location set on targeted rune
GQ (s / nothing) Perform various tasks relating to the GM Queue
    NEXT: Goes to the next call in the queue.
    CLEAR: Clears the call you are currently on.
    CURR: Goes to the call you are currently on.
    If nothing is specified, will display the GM Queue.
 
H
HEAL (Target Command) Fully heals targeted character
HIDE Permanently hides targeted character
HOUSEICONS (on / off) Enables / Disables the display of houses as deeds
HOWTO (s) Opens a gump explaining how to use specified command
 
I
IWIPE (d d d d / nothing) Deletes all NPC's and items not inside a specified square
ITEMMENU (d) Opens specified itemmenu
INCX (Target Command)(d) Increments the X value by the specified amount
INCY (Target Command)(d) Increments the Y value by the specified amount
INCZ (Target Command)(d) Increments the Z value by the specified amount
ISTATS Displays a gump with information on the targeted item
 
J
JAIL (Target Command) Jails the targeted character
 
K
KILLALL (d text) Kills specified percentage of all NPCs and broadcasts text
KICK (Target Command) Disconnects targeted player from the server
KILL (Target Command)(s (h)) Kills character or removes specified item from character
    CHAR: Kills the targeted character.
    HAIR: Removes character hair.
    BEARD: Removes characters beard.
    PACK: Removes character backpack.
    LAYER: Removes specified layer.
 
L
LOADDEFAULTS Resets the default uox.ini settings
 
M
MEMSTATS Opens a gump displaying useful information on UOX's memory usage
MINECHECK (d) Set the server mine check status (0: everywhere, 1: only on cave floors and mountainsides)
MUTE (d / nothing) Squelches specified player for specified amount of time, if any.
MIDI (d / d d) Plays the specified midi file.
MAKE (Target Command)(s) Changes status of the targeted character
    GM: Turns character into a GM.
    COUNSELOR: Turns character into a Counselor.
    PLAYER: Turns character back into a normal player.
MAKESHOP (Target Command) Turns specified NPC into a shopkeeper (giving them buy/sell/bought layers
MANA (Target Command) Restores targeted characters mana to full.
MARK (Target Command) Marks targeted rune to your current location.
MOVETOBAG (Target Command) Moves targeted item into your backpack.
 
N
NACCT (s / nothing) Account Management
    ADD: Brings up Add Account Gump
    BAN: Brings up Ban Account Gump
    EDIT: Brings up Edit Account Gump
    VIEW: Brings up View Account Gump
    LIST: Brings up a Gump with a listing of all Accounts
    If nothing specified, lists all Accounts.
NPCRECT (Target Command)(d d d d) Sets bounding box for targeted NPC.
NPCCIRCLE (Target Command)(d d d ) Sets bounding circle for targeted NPC.
 
O
OPENBANK (Target Command) Opens players bank box.
OPENLAYER (h / nothing) Opens specified layer on character, or the backpack if none specified
 
P
PDUMP Displays some information on UOX's performance
POLY (h / h h) Polymorph yourself into specified creature body.
POST Displays the current bulletin board posting mode for the player
 
R
RECALL (Target Command) Teleports your character to the location marked on the targeted rune.
RELEASE (Target Command) Releases targeted character from jail.
REMOVE (Target Command) Deletes the targeted object.
RESURRECT (Target Command) Resurrects the targeted Ghost from the dead.
REMOVESHOP (Target Command) Turns the targeted shopkeeper back into a normal character.
RESEND Updates all items and characters on your screen.
READLOADINI Reloads the uox.ini.
RELOADJSFILE ( d ) Reloads a specified JScript, based on the scriptID from jse_fileassociations.scp.
RESTOCK (s / nothing) Forces a manual vendor restock.
    ALL: Restock all shopitems to their maximum amount.
    If nothing specified, will do a normal restock.
RESPAWN Forces all spawners in the world to spawn new objects.
REGSPAWN (s / d) Forces spawnregions to spawn new objects.
    ALL: Will respawn in all spawnregions in the world.
    Otherwise the region number to spawn must be provided.
RELOADACCOUNTS Reloads accounts from the accounts file.
REPORTBUG (text) Writes out a bug report to a file.
RELOADDEFS Reloads all dfns from their files.
 
S
SETPOST (s) Sets bulletin board posting type.
    GLOBAL: All Bulletin boards will show the next message posted by you.
    REGIONAL: All bulletin boards in the region will show the next message posted by you.
    LOCAL: Only this board will show the message.
SETRACE (d) Changes your character to the selected race.
SPAWNKILL (d) Kills all characters spawned by specified spawnregion.
SETSHOPRESTOCKRATE (d) Sets the restock rate of all shops in the world.
SECONDSPERUOMINUTE (d) Sets the number of real-world minutes that pass for each UO minute.
SET (s) Sets various attributes on targeted character.
    Base Objects:
      NAME (s): Sets the name of the object.
      STR (d): Sets the strength of the object.
      DEX (d): Sets the dexterity of the object.
      INT (d): Sets the intelligence of the object.
      FAME (d): Sets the fame of the object.
      KARMA (d): Sets the karma of the object.
      KILLS (d): Sets the kills of the object.
      COLOR (h): Sets the hue of the object.
      OWNER (Target Command): Sets the owner of the object.
      X (d): Sets the X coordinate of the object.
      Y (d): Sets the Y coordinate of the object.
      Z (d): Sets the Z coordinate of the object.
      ID (h): Sets the ID of the object.
      VISIBLE (d): Toggles the objects visibility (0: Visible, 1: Hidden, 2: Magically Invisible)
    Characters:
      ALLSKILLS (d): Sets all of the characters skills.
      ORGSKIN (h): Sets the original skin of the character.
      FONT (d): Sets the font type of the character.
      SPATTACK (d): Sets the SPAttack value of the character.
      SPDELAY (d): Sets the SPDelay in minutes for the character.
      POISON (d): Sets the poison strength of the character.
      TITLE (s): Sets the title of the character.
      NPCWANDER (d): Sets the NPCWander type of the character.
      DIR (d): Sets the direction of the character.
      NPCAI (d): Sets the NPCAI type of the character.
      ADVOBJ (d): Set the advancement type the character has used.
      INVULNERABLE (d): Toggle characters invulnerability (0: not invul, 1: invul)
      SPLIT (d): For splitting NPCs, amount of creatures to split into.
      SPLITCHANCE (d): For splitting NPCs, chance of splitting on hit.
      COMMANDLEVEL (d): Set the command level of the character.
      CANTRAIN (d): Toggle characters ability to train (0: can't train, 1: can train).
      FROZEN (d): Toggle characters ability to move (0: not frozen, 1: frozen).
      <SKILLNAME> (d) If a valid skill name is used, sets the specified skill.
    Items:
      AMOUNT (d): Sets the amount of an item.
      MOVABLE (d): Toggles items ability to be moved (0: not movable, 1: movable).
      WIPABLE (d): Toggles items wipeable status (0: not wipable, 1: wipable)
      BUYVALUE (d): Sets items buy value.
      SELLVALUE (d): Sets items sell value.
      RESTOCK (d): Sets amount to restock on a shopkeeper.
      MORE (h): Sets items "more" value.
      MOREX (h): Sets items "morex" value.
      MOREY (h): Sets items "morey" value.
      MOREZ (h): Sets items "morez" value.
      MOREXYZ (h h h): Sets items "morex", "morey", and "morez" values.
      DEVINELOCK (d): Toggles items devine lock status (0: not locked, 1: locked).
      NAME2 (s): Sets items name2 value (For use with itemid).
      TYPE (d): Sets items type.
      DECAYABLE (d): Toggles items decay status (0: not decayable, 1: decayable).
    Spawners:
      SPAWNSECTION (s true/false): Sets the script section for spawner to pull from.
      MININTERVAL (d): Sets the minimum spawn interval.
      MAXINTERVAL (d): Sets the maximum spawn interval.
SFX (h / h h) Plays the specified sound effect.
SHOWTIME Displays the current UO time.
SETTIME (d d) Sets the current UO time in hours and minutes.
SHUTDOWN (d) Initiates server shutdown sequence, parameter is how many minutes until shutdown.
SAVE Forces a worldsave into the .wsc files.
SKIN (h) Changes the hue of your skin.
STATUS Opens the HTML status information gump.
SHOWIDS Display the serial number of every character on screen.
SETSCPTRIG (Target Command)(h) Sets the script trigger number of the targeted object.
SHOWSKILLS (Target Command)(d) Opens a gump displaying the skills of targeted character.
    If the parameter is 0, will use base skill and display only those skills greater than 0.
    If the parameter is 1, will use base skill and display all skills.
    If the parameter is 2, will use actual skill and display only those skills greater than 0.
    If the parameter is 3, will use actual skill and display all skills.
SHOWDETAIL (Target Command) Displays map info on a tile.
SETPOISONED (Target Command)(d) Set the level of poison a character is infected with.
SELL (Target Command) Opens the sell shop gump of targeted shopkeeper.
STAMINA (Target Command) Rejuvinates targeted characters stamina to full.
SQUELCH (Target Command)(d / nothing) Squelch player for given or default amount of minutes.
 
T
TELE (Target Command) Teleports you to the targeted location.
TELESTUFF (Target Command) Moves targeted object to location specified with a targeting cursor.
TELL (d text) Sends a message to the user logged into the specified slot
TEMP Displays the current temperature
TILE (h d d d d d / h ) Tiles the specified item over a square area
    If coordinates are specified, will tile into a box inside those coordinates.
    Otherwise, if only an item ID is specified, will bring up a targeting cursor to create a box.
TIME Displays the current im-game time
TRAVEL Shortcut for opening ITEMMENU 6000 - which is the GM Travel-Menu
TWEAK (Target Command) Opens the "tweak" menu allowing you to modify many settings on an object.
 
U
UNHIDE (Target Command) Shows a player who has been permanently hidden.
UNGLOW (Target Command) Removes the glowing attribute from an object that was made to glow.
 
V
VALIDCMD Displays all valid commands.
 
W
WHO Displays a list of all online players and what slot they are connected on.
WHOLIST (on / off) Opens a gump of all online or offline players.
WIPE (s / d d d d ) Wipes objects based upon passed parameters.
    ITEMS: Wipes all items on the ground in the world.
    NPCS: Wipes all NPCs in the world.
    ALL: Wipes all items on the ground and NPCs in the world.
    If coordinates are specified, will wipe everything inside those coordinates.
WHERE Displays your x, y, z, display z coordinates and world number.
WSTATS (Target Command) Displays information on an NPC's wander mode.
 
X
XTELE (d / h h h h / nothing) Teleports a player to you.
    If you specify a serial number, teleports player with that serial to you.
    If you specify a single number, teleports player logged into that slot to you.
    If you specify nothing, brings up targeting cursor to target player to tele to you.
XGATE (d) Opens a gate to specified location from locations.dfn.
XGOPLACE (d) Sends targeted character to specified location from locations.dfn.
XGO (Target Command)(d d d) Sends targeted player to specified coordinates.
 
Z
ZEROKILLS Resets all players kills back to 0.
